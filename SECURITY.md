## Reporting a Vulnerability

If you discover a security vulnerability in OTPView, please **do not** open a public GitHub issue. Report it privately so it can be addressed before public disclosure.

**Email:** pinakcodestudio@gmail.com / solankimehuln@gmail.com

Please include:
- A description of the vulnerability
- Steps to reproduce it
- The potential impact
- Your suggested fix (if any)

You will receive a response within 72 hours. Once the vulnerability is confirmed and fixed, a new release will be published and the issue disclosed publicly with credit to the reporter.
